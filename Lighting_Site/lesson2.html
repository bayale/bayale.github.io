Intro:

This lesson explores the core properties of light in a stage setting: its color, intensity, and direction. Understanding these elements will help you create moods, highlight performers, and shape the visual experience for your audience.

Color Temperature

Light has a measurable color temperature, expressed in Kelvin (K). Lower values (e.g., 3200K) are warmer and more orange/yellow, while higher values (e.g., 5600K) are cooler and more blue. Matching or contrasting color temperatures can drastically alter a scene’s feel.

Example: Color temperature scale from warm to cool.

Color Mixing

  • Additive Mixing - Combining different colored light sources to create new colors. For example, the RGB (Red, Green, & Blue) diodes commonly used in LED fixtures.
  • Subtractive Mixing - Using filters or gels to absorb wavelengths, changing the output color of white light. This can be done by CMY (Cyan, Magenta, & Yellow) color mixing.

Experimenting with color can evoke emotion, suggest time of day, or direct audience focus.

Direction of Light

The position of the light source changes how subjects appear on stage:

  • Front light - Maximizes visibility, reduces shadows.
  • Side light - Adds depth and shape, often used in dance.
  • Back light - Creates separation from the background, can be dramatic.
  • Top light - Suggests natural sunlight, adds texture.

Intensity & Beam Control

Brightness and beam size affect visibility, focus, and mood. Beam angles can be adjusted via the fixture choice, lensing, or accessories like barn doors and top hats.

Practice Activity

Try creating three different “looks” on stage using the same set of fixtures but varying the color, direction, and intensity. Take note of how each change affects the atmosphere.
